Write a single program that will:

  1. Convert the codons in the a group of cds records into Amino Acid letter codes and write them to a file with appropriate FASTA headers. For example the cds records for Gibberellin. Each codon represents an amino acid, which is encoded as a single letter (akin to the A, C, G, and T for DNA.) The codon list and the single letter codes may be found online at wikipedia under the entry Genetic Code.
    This table shows the 20 standard amino acids used in proteins, and the codons that code for each amino acid.
    Ala A GCU, GCC, GCA, GCG Leu L UUA, UUG, CUU, CUC, CUA, CUG
    Arg R CGU, CGC, CGA, CGG, AGA, AGG Lys K AAA, AAG
    Asn N AAU, AAC Met M AUG
    Asp D GAU, GAC Phe F UUU, UUC
    Cys C UGU, UGC Pro P CCU, CCC, CCA, CCG
    Gln Q CAA, CAG Ser S UCU, UCC, UCA, UCG, AGU,AGC
    Glu E GAA, GAG Thr T ACU, ACC, ACA, ACG
    Gly G GGU, GGC, GGA, GGG Trp W UGG
    His H CAU, CAC Tyr Y UAU, UAC
    Ile I AUU, AUC, AUA Val V GUU, GUC, GUA, GUG
    Start AUG Stop UAG, UGA, UAA
    No single-letter code is given for the stop codons. Use 'Z' for the stop codons.
  2. Find and write to a file, global alignments between the amino acid sequences, the protein sequences corresponding to each. record.
  3. As an example apply the program to the Gibberellin cds records. Pass the name of the file with input gibberellin cds on the command line. Pass the name of the output file for protein sequences on the command line. Pass the name of the output file for the alignments to the command line.
