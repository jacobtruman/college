Bioinformatics
Assignment 02

Programs

Write programs that count the numbers of the bases in the records of the files and count the numbers of the base dimers in the records of the files. The choice of input file and the choice of output file should be in the command line as in: 

   ./countCon all_chrs/all.con all.con.output

Write a program for each of the following file types: 

	.1kUpstream
	.con
	.cDNA
	.intergenic
	.intron
	.seq
	.UTR

named: 

	count1kUpstream
	countCon
	countCDNA
	countIntergenic
	countIntron
	countSeq
	countUTR

Do as many of these as you can. However if you design your program well, there should be only one basic program and each of the requested programs should involve only swapping in a module for the type of file.

Sample Output

  • all.con.output
